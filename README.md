# LifeGame with SwiftWasm

https://life-game-with-swiftwasm.netlify.com/

## Bootstrap

```sh
$ npm install
$ npm run start
```

## Development

```sh
$ swift package --package-path LifeGame generate-xcodeproj
$ open LifeGame/LifeGame.xcodeproj
$ npm run start
```
